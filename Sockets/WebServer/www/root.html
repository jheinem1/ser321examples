You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=[number]&num2=[number] -- returns the product of the two numbers
  • /github?user=[username] -- displays a list of the user's repositories
  • /add-username?username=[username] -- adds the username to the list of usernames (must be a valid github username)
  • /get-usernames -- returns a list of all usernames
